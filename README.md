# nvim-lua-config
Lua files for neovim

Clone this repo to ~/.config/nvim/lua/to4kin

Add configuration to ~/.config/nvim/init.lua

```lua
-- Plugins
require("to4kin.plugins")

-- Setup/Configure
require("to4kin.config")

-- General settings
require("to4kin.settings")

-- Key bindings
require("to4kin.keymapping")
```
